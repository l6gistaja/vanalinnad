Allowing XMLHttpRequest from same site

If you are using ...
  • Opera: allow opera:config#UserPrefs|AllowFileXMLHttpRequest
  • Firefox: set security.fileuri.strict_origin_policy to false at about:config
  • Safari: References panel > Advanced > "Show develop menu in menu bar". Then from the Safari menu "Develop" > "Disable local file restrictions"
  • Chrome: Run Chrome executable with a command line flag --allow-file-access-from-files
  • Internet Explorer: wont work with IE < 9 and with heavy security restrictions
