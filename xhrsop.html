Could not load local XML

Your browser should support JavaScript and JavaScript should be enabled.

If you are using this application locally (from your hard drive), then you can temporarily allow XMLHttpRequest from same site to get it working. NB! This is security risk, disable it afterwards!

Allowing XMLHttpRequest from same site
  • If you are using Opera, allow opera:config#UserPrefs|AllowFileXMLHttpRequest
  • If you are using Firefox, set security.fileuri.strict_origin_policy to false at about:config
  • If you are using Safari: References panel > Advanced > "Show develop menu in menu bar". Then from the Safari menu "Develop" > "Disable local file restrictions"
  • If you are using Chrome: Run Chrome executable with a command line flag --allow-file-access-from-files
